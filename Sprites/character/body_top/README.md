# body_top equipment sprites

Drop body_top piece sprites here. Filename = sprite id with .webp / .png extension
(e.g. `iron_plate.webp` → spriteId `iron_plate`). Each piece authored on the
locked 800×800 anchor template (see EQUIPMENT_TEMPLATE.md at repo root).

After dropping a sprite here, run
`node scripts/import_equipment_sprites.mjs` from the repo root to print the
registry + picker block, ready to paste into `mojiworld_game.html`.
