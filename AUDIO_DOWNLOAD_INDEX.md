## Bulk download flow

For each card on the Ludo audio-generator page:

1. **Click on the card body** → expands to the purple state with audio player
2. **Click the ⋯ (more actions) button** in the top-right of the card → menu opens
3. **Click Download** → file lands in `~/Downloads/` named `audio-<truncated-prompt>.mp3`
4. Rename + move per the table below

Ludo names files using the prompt text, so identifying which one is which is easy — match the **prompt preview text on the card** against the **Prompt snippet** column.

## Tip — speed up the download workflow

Since Ludo's cards revert to the compact state when not hovered, the fastest cadence is:

1. **Click the card body** (anywhere on the blue card) → it expands to purple, exposing the ⋯ button in the top-right
2. **Click ⋯** → small dropdown appears with **Download** and **Delete**
3. **Click Download** → file saves immediately to your default Downloads folder
4. Move to next card

You can keep all cards collapsed in a scrollable grid and just systematically click `card → ⋯ → Download` for each. Should take ~10-15 seconds per file, so ~20 minutes for all 71.

After all files are downloaded, the file names in `~/Downloads/` will look like `audio-Single-short-chibi-anime-male-warrior-pain.mp3` etc. Match each one to the table above and move into the right `audio/<folder>/` directory with the target filename.

## Code already references these paths

The game's audio system (already shipped in v0.25.796) lazy-loads everything from these registries — non-fatal if any file is missing:

- `_AMBIENT_FILES` → `audio/ambient/<name>.mp3`
- `_VOICE_FILES` → `audio/voice/<class>_<m|f>_<kind>.mp3` (NOTE: code currently doesn't include the `_m_` / `_f_` infix — I'll update the registry once files land if you want gendered selection; otherwise pick one gender per class)
- `_SKILL_SFX_FILES` → `audio/skill/<name>.mp3`
- Bosses + monsters: code registries don't exist yet. Once you've got files in `audio/boss/` and `audio/monster/`, ping me and I'll wire them up (new `_BOSS_SFX_FILES` + `_MONSTER_SFX_FILES` registries + hooks in `_playBossIntro()` and `hitMonster()`).
