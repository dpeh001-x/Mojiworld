## Automation

Submit order matches the numbered list above. The Chrome MCP driver
pastes each prompt into Ludo's `app.ludo.ai/image-generator` prompt
box and clicks **Generate**. Each generation lands as a card in the
user's queue. After all 19 land, the user picks favourites and
downloads — same `card → ⋯ → Download` workflow documented in
`AUDIO_DOWNLOAD_INDEX.md`.

**Credits:** ~19 (one render each, ~1 credit per image render on Ludo).
Budget 25-30 if you expect to re-roll the highest-risk silhouettes
(Sundered Smith + Hidden Pagoda).

**Downloaded file paths after picking:**
- Monsters → `Sprites/monsters/<id>.webp` (background-removed)
- Boss → `Sprites/bosses/sundered_smith.webp` (background-removed)
- NPCs → `Sprites/npc/<name>.webp` (background-removed)
- Backgrounds → `backgrounds/bg_v3_<mapId>.png` (full bleed)
